A modern, full-stack web portfolio built with a Bento Box grid layout. Unlike a static resume, this application features a dynamic backend for real-time content management.

Core Features

Dynamic CMS: Manage certificates, work experience, and tech stacks via a Node.js/Express backend and MySQL database.
Automated Thumbnails: Integrated PDF.js to automatically generate visual previews for uploaded project documents.
Interactive Sandbox: A dedicated environment for hosting and interacting with live project demos.
Modern UI: A responsive, high-performance interface built with Tailwind CSS.

Tech Stack

Frontend: HTML5, Tailwind CSS, JavaScript

Backend: Node.js, Express.js

Database: MySQL
